浏览器主页劫持修复与Console简单个性设置

发表于 2016-07-26   |   分类于 blog

浏览器主页劫持修复

前几日，不知下载使用了哪个什么软件，意外遇到了某数字安全软件厂商的流氓导航绑架。以前也曾经遇到过，但是基本都是更换浏览器快捷方式或者修改快捷方式启动参数就能解决问题，这次折腾了好久，还是没有解决。遇到打开浏览器就导航到某个网站，我遇到的情况基本就两种，主页篡改和启动参数修改。

  • 主页篡改修复

    遇到这样的情况，基本不用太担心，修改浏览器主页，查杀木马就行。很简单的使用各大安全软件厂商的卫士/管家基本都有修改锁定主页的功能，不需要再费心思修改用户策略表什么的。

  • 启动参数修复

    遇到这样的情况，找到浏览器的快捷方式，右键， 查看属性，一般都可以看到目标栏目下除了执行文件地址还加了一长串的网址，这个网址就是被劫持前往的网址。修改更换快捷方式即可。但是我的chrome浏览器快捷方式后面没有参数，但是在浏览器中使用chrome://version命令后可以看到命令行栏目中加了一长串网址。通过各种办法无法解决，甚至使用了PCHunter查看浏览器进程是否未知名的模块链接注入。查看无果，无奈之下，选择自己定义启动参数来解决劫持问题。在浏览器快捷方式属性-目标栏目中，在”chrome.exe”参数后添加空格和”–”解决，也可以在浏览器快捷方式中添加”–”和常用主页地址。每次启动打开就不会链接到流氓地址了。但是，还存在一个问题就是在已打开的浏览器中任务栏图标右键打开新的窗口时仍然会导航到流氓导航。

在这次与流氓的斗争中，我暂时失败了，甚至想到过使用流氓软件对付它自己这个流氓，无效。还还原了我的许多个性化设这。重新设置的时候，顺便记录一下吧。

CMD命令行在任意位置目录打开

  • 在文件夹右键时显示cmd命令行选项

    进入注册表 运行，输入regedit，找到 \HKEY_CLASSES_ROOT\Directory\shell\ 右键新建项 输入 CommandPrompt，值可以任意设置为你想要显示的名字。如 打开cmd…。右键该项目， 新建项 输入 Command，数值设置为 cmd.exe /k cd “%1” 。

  • 在文件夹空白位置显示cmd命令行选项

    进入注册表 运行，输入regedit，找到\HKEY_CLASSES_ROOT\BACKGROUND\Directory\shell\ 右键新建项 输入 CommandPrompt，值可以任意设置为你想要显示的名字。如 打开cmd…。右键该项目， 新建项 输入 Command，数值设置为 cmd.exe /k cd “%1” 。关闭注册表就即可看到效果啦。

修改CMD的字体

每天都需要用到cmd，可是它那么丑的字体，看着难受还眼睛疼。这时候，可以自定义字体，再也不用看cmd下显示得那么丑的宋体字了。进入注册表 运行，输入regedit，找到\`HKEY_CURRENT_USER\Console\%SystemRoot%_system32_cmd.exe\，点击查看右边有一系列的值，这些值就是cmd的个性化设置啦。找到或者新建字符串值 FaceName，值为你想选用的字体，这里我选择为 Lucida Console。FontSize是自已大小，我选择为十六进制e0000，14号字体大小。也可与进行更多高级的设置，但是对我来说已经足够了。

基本设置就这些了。更多高级的设置，可以上网搜寻更多资料。不过注意注册表记得备份，万一不小心更改了某些数值还忘记了，找不回来就悲剧了哦。
